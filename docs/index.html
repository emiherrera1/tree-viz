About This Visualization

What is this?

This is a radial tree visualization of the Content Terminal Niche Classification Standard (CTNCS). It shows the hierarchical structure of content categories and their relationships.

How to Navigate

  • Zoom: Use the +, -, and Reset buttons to zoom in/out
  • Pan: Click and drag to move around the tree
  • Search: Use the search bar to find specific nodes
  • Click Nodes: Click any node to see its path and details

Depth Controls

Use the "Show Levels" buttons to control how many levels of the tree are visible:

  • 1-2: Shows only the main categories (cleanest view)
  • 1-3: Shows main categories and subcategories
  • 1-4: Shows deeper levels of detail
  • All: Shows the complete tree (may be crowded)

Understanding the Tree

  • Green Center: Root "Content" node
  • White Circles: Category nodes
  • Gray Lines: Connections between categories
  • Text Labels: Category names

About CTNCS

The Content Terminal Niche Classification Standard is a comprehensive taxonomy for understanding content categories and their relationships. It helps classify and organize the vast landscape of digital content.
